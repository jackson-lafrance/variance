# React Portfolio

![fed-portfolio.png](fed-portfolio.png)

## Setup
- Clone the repository to your local machine
- Install all dependencies with `yarn`
- Load the application into your browser with `yarn start`

In this assignment we'll use React & TypeScript to create a simple Portfolio website. You may create a portfolio for yourself, or, as a twist you may choose any fictional character you like to create a Portfolio for. Information in the Portfolio should at least include a description of yourself or the character, past feats / accomplishments with descriptions, and images.

### Minimum Requirements
- A single page React webapp
- All JSX is properly semantic
- Code is separated into components
- Portfolio contains at least:
  - A description of the character
  - A list of past feats / accomplishments with descriptions
  - Images of the character
